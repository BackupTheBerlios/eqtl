This directory contains all files required to erect the web interface
for the browsing of expression QTL for EAE - except - the eQTL data itself.

eisen.txt	- expression data for all mice
expression_trait_correlation.php
		- web interface to determine degree of correlation
		  between expression data and classical traits
footer.php	- end of web pages
func_configuration.php
		- helper to set up basic parameters
func_connect.php
		- helper to connecto to databases
func_conversion.php
		-

		-

		- helper to configure the accession of all databases
func_expressionview.php

		  from EnsEMBL
header.php	- header of web pages
iHopInteraction.php
index.html	- referral to php page
index.php	- start page
interaction.php - retrieval of epistatic effects between Loci
locus.php	- retrieval of loci associated to the disease
map.sql		- 
marker.php	- retrieval of marker positions
mouse_scores.csv
		- classical scoring data
parental_expression.php
		- retrieval of expression data from parental strains
phpinfo.php
		- test of PHP installation
pzr.css
		- style sheet
qtl.php
		- retrieval of direct association of loci with
		  disease scores
trait.php
		- retrieval of information from expression QTL genes
trait.sql
		- description of trait table in database
trait_upload.pl
		- script to upload trait information from Illumina
		  chip descriptions
